# Chatbot Automation System

This project is a web-based chatbot system developed using Flask and SQLite.

## 🚀 Features
- Intent-based response system
- Admin panel for managing data (CRUD)
- SQLite database integration
- Automated response generation

## 🛠 Technologies Used
- Python (Flask)
- SQLite
- HTML/CSS

## 📌 Purpose
The goal of this project is to automate response systems and reduce manual interaction.

## ⚙️ Setup
1. Install requirements
2. Run the Flask app
3. Access via browser
